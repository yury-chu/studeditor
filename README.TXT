Приложение по тестовому заданию.

Среда разработки:
Ubuntu 14.04 LTS
Google Chrome
PyCharm 4.5.3

Python 2.7.6
Django 1.8.3
Бд Sqlite3.

Необходимые модули прописаны в requirements.txt
(Содержит модуль необходимый для перевода проекта на Postgresql)

1)На данный момент не реализован поиск, и сортировка по имени и т.п.


Реализовано:
-Авторизация посредством встроенной системы авторизации Django
-Базовый уровень ограничения доступа
-Добавление групп.
-Добавление студента.
-Отображение списка студентов: 
	Администратор super пароль super - видит все записи
	Остальные пользователи видять только те записи, которые они добавили.
-Небольшое визуальное оформление


Разрабатывал с пониманием того, что фреймворк django знаю не очень уверенно.
Поэтому не использованы оптимальные для данных задач модули.
В связи с чем пока сделал как смог, на основе имеющихся знаний.

Работу с админкой не вел. (По условиям из ТЗ)
